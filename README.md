# Aarunya Healthcare System

A collaborative MERN-stack healthcare platform for doctor appointment booking and AI-assisted healthcare support.

## My Contributions
- Designed and implemented the Helpdesk chatbot interface using React.js.
- Integrated the chatbot with backend REST APIs and Gemini 1.5 Flash API.
- Implemented asynchronous loading states and improved user interaction.
- Debugged frontend-backend integration issues.
- Resolved Git merge conflicts during collaborative development.
- Enhanced chatbot responsiveness and overall user experience.
